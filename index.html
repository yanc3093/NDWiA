Next-Day Weather in Australia

About this project

This project was an individual school assignment completed in April 2022.

I was provided with a dataset containing about 10 years of daily weather observations from many locations across Australia. The objective of this project was to build different classification models that could predict whether tomorrow would be warmer than today or not.

Project outcomes

As usual, I started the project with data exploration and preprocessing.

After that, I built 6 different classification models using the respective functions provided by the R libraries:

  • Decision Tree: "tree"
  • Naive Bayes: "e1071"
  • Bagging and Boosting: "adabag"
  • Random Forest: "randomForest"
  • Artificial Neural Network: "neuralnet"

To measure the models' performance, I computed the prediction accuracy as well as the Area Under the Curve (AUC). Below is a screenshot of the result.

Besides, I also plotted the Receiver Operating Characteristic (ROC) curve for each classification model.

Based on the observations above, I finally concluded that the Artificial Neural Network is the "best" classifier among all the models for this use case.
